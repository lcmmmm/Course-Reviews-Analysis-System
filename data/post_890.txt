      哪一學年度修課：107-2

      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)蔡欣穆

      λ 開課系所與授課對象 (是否為必修或通識課 / 內容是否與某些背景相關) 
        資工系選修
      δ 課程大概內容
        熟悉網路/系統管理的背景
        介紹常用工具
        1.System Adminstration Overview
        2.IP Protocol Stack
        3.UNIX and Shell Command
        4.Link Layer and Physical Layer
        5.Storage
        6.Advanced Link Layer
        7.Virtualization
        8.IP
        9.Security
        10.Kernel
        11.Package Management
        12.IP Services
        13.NFS
        14.Wireless
        15.Web Hosting

        Lab(實驗):
        0.Ethernet Cable
        1.Shell Script
        2.TCPDump and IPerf
        3.Partition and Filesystem
        4.Cisco Packet Tracer
        5.Docker
        6.pfSense
        7.Security(SYN Cookie)
        8.DNS
        9.System Configuration
        10.Wireless
        11.Web server and SQL

      Ω 私心推薦指數(以五分計) ★★★★★
        本系學生:想要成為資工系的管理者★★★★★
        外系學生:想要學到各種實用技能★★★★★
      η 上課用書(影印講義或是指定教科書)
        參考用書:1.UNIX and Linux System Administration Handbook(4th Edition)
                 2.The Practice of System and Network Administration(2nd Edition)
      μ 上課方式(投影片、團體討論、老師教學風格)
        使用投影片,Zuvio,團體討論，老師會和同學互動
      σ 評分方式(給分甜嗎？是紮實分？)
        35% 作業(5%*7)
        20% 期中考
        30% 期末考
        15% 課程參與(課程上的操作和出席率)

        個人覺得是紮實甜，作業量對於不習慣查資料或實作的可能會很辛苦
      ρ 考題型式、作業方式
        每兩周繳交一份作業(使用NTU COOL)，每個單元的後面都會有TA來帶實驗課程
        分成期中期末兩次，都是上機，採取破關->得分的方式，每次三小時，每組三個人
        可以上網查資料和和隊友討論，配分網路和系統各半
      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)
        出席率非常重要，會用Zuvio來點名(簽到)，課堂上的Zuvio問題還有第三節課的Lab也會算成績
        平常會無聊自己玩系統或看相關資訊的會輕鬆很多
        另外最好要會使用Google查資料或去Stack Overflow上面發問
        老師上課很有趣，基本上不要在上課時間聊天，不過有問題還是要和一下身旁的同學討論比較好

        另外這門課在寒假的時候會在老師的網站上面放出HW0，要早點開始寫
        還有因為我修的時候人太多，所以門檻很高，外系扣超過1分就沒辦法簽了
        本系也要在扣5分以內(以上總分皆包含加分題)

      Ψ 總結
        我覺得這門課很有趣，因為會介紹到一些很實用的工具，加上有很多實作應用，
        相對於單純理論來說，這個可能比較容易得到成就感
        不過要修這門課要有幾個心理準備，如果不想每份作業寫10小時以上的話最好不要修
        還有要找好大腿(X，在期中期末的時候抱著大腿就能輕鬆拿高分
        或是要夠熟悉上課提到的工具，老師不會在上課直接教這些工具，要學好還是要
        認真寫作業
        有問題可以去問TA，因為TA都是資工系NASA團隊的現役成員，所以可以幫很多忙
        關於資安的部分會請本系的蕭旭君老師來上一部份，會學到一些實務上會碰到的問題

        本系學生修這門課可以選擇抵免服務學習(但是就沒有系選修的2學分)，但還是
        會有實驗課程的1學分，另外如果成績夠高，還可以進入二階和三階課程，實際成為
        管理團隊
        外系學生就沒辦法進入二階，但是還是可以從中學到很多東西
        總體來說，大推這門課
