      哪一學年度修課： 104-2

      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)
         謝尚賢教授

      λ 開課系所與授課對象 (是否為必修或通識課 / 內容是否與某些背景相關) 
         土木系大一必修

      δ 課程大概內容
         就開給初學者的程式設計，程式語言為JAVA，以下為各周課程內容：
          1. Course Introduction Introduction to Programming and Java
          2. Elementary Programming, Selection
          3. Selection, Math functions, Characters and Strings
          4. Loops
          5. Methods
          6. Single-dimensional Arrays
          7. Multi-dimensional Arrays
          8. Intro to Objects and Classes
          9. Objects and Classes
         10. Object Oriented Thinking & Strings
         11. Inheritance and Polymorphism
         12. Abstract Classes and Interfaces
         13. JavaFX Basics, and Event-Driven Programming and Animations
         14. JavaFX UI, and Graphics

      Ω 私心推薦指數(以五分計) ★★★★★
         ★★★★★

      η 上課用書(影印講義或是指定教科書)
         Y. Daniel Liang (2015), Introduction to Java Programming,
         Comprehensive Version (10th Edition), Prentice Hall

      μ 上課方式(投影片、團體討論、老師教學風格)
         土木系大一有開三個班的計算機程式，是統一教學，講義、小考、期中期末考，
         都是一樣的資料，不過這個班是採用「翻轉教室」的方式教學，老師有先預錄
         整學期的影片(一周的影片量不會超過2小時)，希望是上課前大家就已經先看過
         ，在上課時就可以直接上機練習，只要當周的lab題目過了，就可以下課，
         當然也是可以在課堂中才看影片；另外老師也有說如果不習慣這種教學方式，
         可以去另外兩班聽課，lab跟考試時在原班就好

         寫lab題目時，老師會巡教室看大家寫程式的情況，如果有問題都可以直接問
         老師或助教

      σ 評分方式(給分甜嗎？是紮實分？)
         Lab實作20%、3次小考共40%、期中考15%、期末考25%

         印象中第一次小考班平均是45分，有調分把平均變60，但不確定後面幾次
         有沒有調整；期末考滿分是給140分

         最後看學校成績系統的資訊大概有40%的人A+

      ρ 考題型式、作業方式
         小考就都當周的第一節課上機考
         期中有手寫的部份，但佔比不大，大多還是上機實作的分數，期末好像沒有手寫

         每周Lab實作大概3題以內，另外每周也會有homework，但不需要繳交

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)
         沒點過名，但因為交Lab作業是要在現場compile給助教看，
         所以若要Lab分數就還是不能翹課

         因為是大一的課，不需要基礎

         加簽方面，由於上課會需要用到電腦，所以一定有人數限制，如果是自己系上
         已經有開計算機程式之類的課，老師不會給授權碼，其他外系若想加簽要提供
         為何想選修這門課的原因，印象中加簽到的人數應該不超過5人

      Ψ 總結
         我個人蠻喜歡這種上課方式的，影片可以調速度，大概都一個多小時就看完，
         若有聽不懂的地方，也可以隨時再拉回去看

         老師跟助教都很有教學熱忱，有問題在課堂中或課後寫信問都可以

         由於沒修過其他系開的大一計程，所以也無從比較難易度及課程好壞，
         但身為沒碰過程式的人，個人認為這門課的課程安排是蠻適合程式初學者的，
         涵蓋不少範圍(不過沒教I/O就是了)，也沒有講到太深，
         而JAVA本身也比C++容易上手
