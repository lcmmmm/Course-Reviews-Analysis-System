      哪一學年度修課： 111-2

      ψ 授課教師 （若為多人合授請寫開課教師，以方便收錄）

      劉立偉 許聿廷

      λ 開課系所與授課對象 （是否為必修或通識課 /內容是否與某些背景相關）

      土木系大一必修

      δ 課程大概內容

      建立一個供小球（如鋼珠）運行的軌道模型（兩條軌道，分別給不同的兩種小球走
      ），結合其他土木系必修課（應用力學2 、計算機程式、微積分），分析小球的運
      動情形。分為結構課、自控課，只有負責該課的老師會來，另外還有幾周是兩個老
      師都會來（通常是大報告）

      Structure         1. 2D track fabrication
      Automation        2. Arduino （LED, bread board, simulator）
      Structure         3. 2D roller dynamics modeling using discretized ODE
      Automation        4. Sensor, read measurements
      Structure         5. Review and clinic: comparison of 2D roller model
                           with experiment
      Joint             6. Midterm Project Evaluation
      Joint             7. Final project brainstorming/3D track fabrication
      No Class          8. Midterm week: No Class
      Automation        9. LEGO & Motor
      Structure         10. 3D track geometry and fabrication
      Automation        11. Wireless Automation
      Structure         12. 3D roller dynamics modeling and fabrication
      Automation        13. Automation trouble shooting
      Structure         14. Structure simulation trouble shooting: comparison
                            of 3D roller model with experiment
      Joint             15. Final Project Evaluation
      No Class          16. Final exam week: No Class

      Ω 私心推薦指數（以五分計） ★★（但我同學都給★★★）

      喜歡動手實作、烙鐵達人                           ★★★★★
      手拙的人                                         ☆
      第一次接觸程式是從系上的必修計程的人             ★★
      喜歡力學分析、推理的人應該會喜歡這堂課，不過得看教授

      η 上課用書（影印講義或是指定教科書）

      無（但基本上會需要用到應用力學2的課本）

      μ 上課方式（投影片、團體討論、老師教學風格）

      結構課 劉立偉 授

      建立一個供小球（如鋼珠）運行的軌道模型（兩條軌道，分別給不同的兩種小球走
      ），先報告（如果有要報告的話），接著是老師講解當周課程內容（一些力學推導
      ），通常會提早講解完，接著就是自由時間，可以留下來和老師、學生助教討論或
      是到S-lab 繼續做軌道或是直接離開。老師講解的步調有點快，聽其他組說，他們
      其實跟不太上。但由於我們組在課堂前常常已經想過當周的內容了，到後期，基本
      上就只是老師把我們已經有想到的東西再講一遍。

      自控課 許聿廷 授

      因為結構課的負擔很重，所以自控課比較輕鬆，通常是上課後老師開始講解如何用
      Raspberry Pi Pico 板以不同的方式接線，來達成不同的功能，進而使用在我們的
      軌道上。最終的目標是要能分辨兩種不同的球，進而使其運行在它應該運行的軌道
      上。會提早約一個小時下課，可以留下來和老師、學生助教討論或是到S-lab 繼續
      做軌道或是直接離開。

      σ 評分方式（給分甜嗎？是紮實分？）

      扎實看各組、不甜，但基本有完成軌道、報告，大概會有A-以上。
      （不過成績分布還沒出來，我不敢肯定）
      目前看來，名字以土木工程開頭的課都有這種特性。

      ρ 考題型式、作業方式

      期中要完成2D軌道，期末要完成3D軌道，分別有一次大報告，那一次上課同時會完
      成小球在軌道上運行的某些檢測（如預測運行時間、脫軌點等）。小報告的要求不
      是很明確，有時候會不小心做到下次上課的內容，導致上課的時候會覺得在浪費時
      間；程式的檔案會分批給，一樣，這次可能想很久的部分，下次又會直接出現在範
      例程式裏。

      ω 其它（是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
      加簽習慣？嚴禁遲到等…）

      加簽
      因為時段的關係，修的人蠻少的，本系（含雙修）的應該都會簽。

      老師個性
      上傳簡報務必準時，否則劉老師可能不會讓你現場繳交PPT。

      課後做軌道時段預約
      期中過後開始製作3D軌道，這個時候大助教會寄信詢問各組要來S-lab 的時段（平
      日晚間及假日，平日白天要使用S-lab 直接找黃技師即可），預約後會點名，但沒
      有預約的人還是可以去（但不保證有位子）。之後視情況還會有第二次的調查，但
      根據我的經驗，因為一次調查就是一兩個禮拜的時間，各組根本不會知道自己做不
      做的完（如果能做完就不需要預約，預約了人還要到，不然會被扣分），所以第二
      次調查基本上沒什麼人填，有填的那幾天，實驗室就會爆滿。

      建議可以改成安排助教輪守實驗室，分別負責固定的時間，公告預約實驗室的表單
      連結，在前一天收單，如果都沒有人填寫，則當天不開放。

      組員
      建議在選課之前就找好組員，好的組員帶你飛上天，有修過土木工程概念設計的人
      應該都知道。

      Ψ 總結

      這是一門兩學分，但平均一周（三四個人各自）可以花掉十小時的課。（當然，情
      況依組別而定，如果是手做比較好的人或是製程較佳的組別，不會這麼長。）從期
      中後開始，一周至少兩天以上下課後要往 S-lab跑，軌道的製作跟土木的相關性不
      大，但六成以上的時間都是花在製作及處理軌道上。除了做軌道本身很難以外，當
      軌道接上去後，還要花很長的時間調整（短則幾分鐘，長則數小時）以讓小球能順
      利運行。我可以感覺到，老師們很想把系上必修與這堂課做結合，但是同學大部分
      的時間是花在做軌道上，而不是思考，建議改成只做一條軌道。

      力學的推導應該是這堂課的精隨（應用力學要好好上課），尤其是從2D推廣到3D之
      後，但課堂上的內容不一定能模擬出和實驗數據相似的結果，到了後期（其他班次
      的）很多組都在硬湊數據。而我們組主要靠組員絞盡腦汁想出除了幾個基本的作用
      力外，其他影響小球運動的因素。假設的合理性不是那麼重要，教授們較注重實驗
      數據和模擬數據貼近的程度。

      程式的部分，一組裡面至少要有一兩個對程式稍微有點基礎的人，不然整學期都會
      很難過，本系開的計算機程式的進度無法與土木工程基本實作配合。

      即使如此，還是要感謝學生助教的鼎力相助，在我們遇到瓶頸的時候，不管是力學
      分析或者是自控，提供建議給我們，幫助我們能夠更迅速的解決問題。

      總而言之，土木工程這四字開頭的課都很厚工。
