      哪一學年度修課：
        109-1
      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)
        蔣明晃
      λ 開課系所與授課對象 (是否為必修或通識課 / 內容是否與某些背景相關) 
        工管系、商研所

      δ 課程大概內容
        from CEIBA

        09/14   Introduction, Brief review of linear algebra
        09/21   Linear Programming (Simplex Method)
        09/28   Linear Programming (Simplex Method)
        10/05   Goal programming & Sensitivity Analysis
        10/12   Revised Simplex & Sensitivity Analysis (HW#1 due)
        10/19   Duality
        10/26   Duality
        11/02   Transportation Models, Assignment Models, & Transshipment Models
        11/09   Network Models (HW#2 due)
        11/16   Midterm
        11/23   Network Models
        11/30   Integer Programming (HW#3 due)
        12/07   Nonlinear Programming
        12/14   Nonlinear Programming
        12/21   Nonlinear Programming (HW#4 due)
        12/28   Deterministic Dynamic Programming
        01/04   Deterministic Dynamic Programming
        01/11   Final Exam

      Ω 私心推薦指數(以五分計) ★★★★★
        ★★★★★ 我還不推爆
      η 上課用書(影印講義或是指定教科書)
        Winston, W. L., Operations Research: Applications and Algorithms, 4th
        edition, Thomson Learning
        網路找得到書，圖書館也有書

      μ 上課方式(投影片、團體討論、老師教學風格)
        上課投影片講述，老師會用ipad寫計算過程給同學看，有時候會問一些觀念題
        給同學加分

      σ 評分方式(給分甜嗎？是紮實分？)
        20% 作業(四次)
        40% 期中考
        40% 期末考

        老師都說他給分是為了鼓勵學生，但看結果是甜到蛀牙，常常拿到意想不
        到的分數

      ρ 考題型式、作業方式
        作業為課本習題，如果沒有複習的話其實不好寫
        考試為開書考，主要會小調整一些地方，來確認同學觀念是否清晰

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)
        老師會希望大家先修管科模，但其實不用也沒差，真正要的先備知識就是高中
        程度的矩陣運算，然後微積分(主要是微分)

      Ψ 總結
        這門課可以說是作業研究的基石，介紹各種最佳化的演算法，了解各個演算法
        背後的意義，老師說得鉅細靡遺，考前office hour也對學生很有耐心，真的是
        難能可貴的好老師。老師都會說這門課不會太難，因為他不是要叫你發展演算法，
        而是了解背後意義即可，剩下就是step by step去執行。
        另外，程式撰寫的部分也不是本門課程的重點，要coding應用的話，可能要去找
        小傑或是家岩老師了。

        很感謝自己期初有選這門課，不然我這學期真的不知道在上什麼碗糕，總之，
        蔣老師真的不愧為供應鏈演算法大師，同學下課也可以去找老師聊天，會很
        有收穫的！推推老蔣～～
