      哪一學年度修課：
        106-2

      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)
        蕭旭君老師

      λ 開課系所與授課對象 (是否為必修或通識課 / 內容是否與某些背景相關) 
        資工所，網媒所

      δ 課程大概內容
        第一週：  Course Information
                  介紹沒簽到的同學如何加簽（hw0），給四篇paper挑一篇
                  來寫 critique 。第二週公佈錄取名單並發放授權碼。
        第二週：  Crypto and Security Overview
        第三週：  Cryptographic Hash Function
        第四週：  Symmetric-key Cryptography
        第五週：  Public Key Cryptography/Public Key Infrastructure
        第六週：  No Class
        第七週：  Entity Authentication
        第八週：  Anonymous Communication
        第九週：  1st midterm
        第十週：  BGP/(D)DoS
        第十一週：DDoS/Transport Layer Security
        第十二週：Transport Layer Security
        第十三週：Guest Lecture: IoT Security
        第十四週：Web and Software Security
        第十五週：Cryptocurrency Technologies
        第十六週：2nd midterm
        第十七週：Group Presentation - I
        第十八週：Group Presentation - II

      Ω 私心推薦指數(以五分計) ★★★★★
        這堂課Loading不輕，第一次作業截止後少了十來個人～ＯＡＯ
        做好心理準備的人：五顆星～

      η 上課用書(影印講義或是指定教科書)
        沒有上課用書。上課材料以及複習材料是老師的上課用投影片，
        但是老師的投影片以圖片居多，最好還是上課邊聽邊記比較有印象。

      μ 上課方式(投影片、團體討論、老師教學風格)
        上課是用投影片，老師貌似每年都會修改一下的樣子。
        基本上是照著投影片去上，但是老師口條順暢，所以應該不會覺得無聊。
        老師下課也樂於回答同學的提問。

      σ 評分方式(給分甜嗎？是紮實分？)
        作業            40%
                                - Labs(3*10%)
                                - Reading critiques（5*2%）
        期中考          2*15%
        期末團體project 25%
        課堂參與        5%

        Labs 每次分兩部分，上半部（通常佔30分）類似名詞解釋；下半部
        是 CTF 形式的題目，幫助同學更了解上課內容。
        第一次 Lab 比較麻煩，要解 cipher 以及熟悉 pwntool ，所以會花
        比較多時間。
        第二、三次在熟悉 pwntool 後就比較輕鬆了。
        作業的部分有問題的話可以寄信問助教，或者把握住助教時間去當面詢問。
        謝謝助教們的耐心回答 <(_ _)>

        Reading Critique 每兩週從老師挑的二至三篇論文中選一篇寫critique。
        論文的篇幅大概落在 4 到 10 頁，通常要花1-2小時來寫。

        兩次期中考都很早考，會錯開大考週，讓同學有時間做 final project。

        Final project是三至四人一組，要出一份小論文（6面或以上，ACM 格式）
        以及上台報告。
        題目只要和密碼學與資安有關都可以。我們這組是用機器學習來給病毒上
        tag，其他組有做台大網站的資安分析、或者破解咖啡廳 wi-fi 等等...
        老師及助教會依完成度來評分，還有組間互評，獎品是老師提供的$$$$$。

        課堂參與的話老師只要對你有印象應該就有 5%。

        總結來說算是扎實，甜不甜就看個人造化了。

      ρ 考題型式、作業方式
        總共有兩次期中考。考前老師會給checklist，內容是所有上課內容
        的標題，照著那個去準備應該可以拿到一定的分數。另外也會考一些上課
        的東東還有reading critique讀過的paper。

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)
        不注重出席，但還是建議出席，這樣才可以了解一下作業該從何下手。
        外系的話建議有一點 python 基礎。不會的話也沒關係，我也是因為這堂課
        才學寫 python 的。

      Ψ 總結
        謝謝老師這學期的教導，另外也謝謝助教們耐心的回答。
        還有謝謝跟我期末 project 同組的學長們的 carry 。
